Course

A collection of all available courses

/course get

get

Get a list of all available courses and the corresponding document count

get /course

Get a list of all available courses and the corresponding document count

  • Response

HTTP status code 200

Body

Type: application/json

Example:

[
    {
        "id": 815,
        "name": "Introduction to advanced examples",
        "count": 1
    }
]

/course/{id}/{type} get

get

Get a single course and all corresponding documents

get /course/{id}/{type}

Get a single course and all corresponding documents

  • Request
  • Response

URI Parameters

  • id: required (integer)

    The ID of the course

  • type: required (string)

    Only return documents of the specified type

HTTP status code 200

Body

Type: application/json

Example:

{
    "id": 815,
    "name": "Introduction to advanced examples",
    "documents": [
        {
            "id": 812,
            "type": "written exam",
            "date": "2011-05-14",
            "lecturer": "Jane Doe"
        }
    ]
}

HTTP status code 404

Body

Type: application/json

Example:

{ "message": "Course not found" }

Order

A collection of orders

/order post

post

Order a print

post /order

Order a print

  • Request
  • Response

Body

Type: application/json

Schema:

{
  "$schema": "http://json-schema.org/draft-04/schema",
  "title": "Order",
  "description": "A print' order",
  "type": "object",
  "properties": {
      "name": {
          "description": "The orderer's name",
          "type": "string"
      },
      "mail": {
          "descrption": "The orderer's e-mail address",
          "type": "string",
          "format": "email"
      },
      "documents": {
          "descrption": "The array of ordered document IDs",
          "type": "array",
          "items": {
              "type": "integer"
          },
          "minItems": 1,
          "uniqueItems": true
      }
  },
  "required": ["name", "mail", "documents"]
}

Example:

{
  "name": "John Doe",
  "mail": "john.doe@example.com",
  "documents": [812]
}

HTTP status code 201

Body

Type: application/json

Example:

{ "message": "Order successful" }

HTTP status code 401

Body

Type: application/json

Example:

{ "message": "e-mail address invalid" }
